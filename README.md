## Getting Started

你可以选择在线阅读我们编写的[配套文档](http://www.tokisakix.cn/2023-Fall-YatCPU-Docs/)

你也可以选择在下载到本地后阅读
```
git clone https://github.com/Tokisakix/2023-fall-yatcpu-repo.git
cd 2023-fall-yatcpu-repo/docs
pip install -r requirements.txt
mkdocs serve
```
